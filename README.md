# Watir test code

##Setup

1) Install [Ruby 2.7.1](https://www.ruby-lang.org/en/documentation/installation/)
1) Instal Watir
```sh
> gem install 'watir'
> gem install 'webdrivers'
> gem install 'rspec'
> gem install 'faker'
```
1) Run a sample program
```sh
> ruby simple_tst.rb
```

1) Run tke tests
```sh
> rspec spec/test_form_view_spec.rb 
```
